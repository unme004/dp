Input n and array p[0…n-1]

Create dp[n][n]

For i = 1 to n-1:
    dp[i][i] = 0

For length = 2 to n-1:
    For i = 1 to n-length:
        j = i + length - 1
        dp[i][j] = ∞
        
        For k = i to j-1:
            cost = dp[i][k] + dp[k+1][j] + p[i-1]*p[k]*p[j]
            dp[i][j] = min(dp[i][j], cost)

Print dp[1][n-1]

Time Complexity : O(n^3)
Space Cimplexity : O(n^2)
